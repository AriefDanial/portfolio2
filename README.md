# Portfolio 2.0

A modern, interactive personal portfolio built with vanilla HTML, CSS, and JavaScript. Features a custom command palette, multi-theme support (including a 🦖 Jurassic mode), smooth animations, and a fully responsive design.

🔗 **Live Demo:** (https://portfolio2-arief.vercel.app)

---

## ✨ Features

- **🎨 Three Themes** — Light, Dark, and a custom Jurassic mode with earthy tones and ambient ferns
- **⌨️ Command Palette** — Press `/` or `Ctrl/Cmd + K` to quickly jump to any section
- **⌨️ Keyboard Shortcuts** — `T` to cycle themes, `1/2/3` for instant theme switching
- **✍️ Typewriter Effect** — Dynamic role text in the hero section
- **🖱️ Custom Cursor** — Smooth follower cursor with hover interactions (desktop)
- **🌿 Animated Noise Texture** — Premium film-grain overlay across the entire page
- **📊 Animated Skill Bars** — Progress bars that animate on scroll
- **🪐 3D Hero Visual** — Interactive orbiting rings with mouse parallax
- **📱 Fully Responsive** — Optimized for all screen sizes

---

## 🛠️ Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, gradients, animations, 3D transforms
- **Vanilla JavaScript** — Intersection Observer, custom events, DOM manipulation
